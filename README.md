### Quickstart

#### Install dependencies

```
npm install
```

#### Run the app

```
npm run start
```

#### Build the app (automatic)

```
npm run package
```

#### Build the app (manual)

```
npm run build
```

#### Test the app (after `npm run build` || `yarn run build`)
```
npm run prod
```
